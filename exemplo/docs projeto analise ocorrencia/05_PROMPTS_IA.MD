# Prompts IA

## Objetivo do documento
Registrar prompts usados no desenvolvimento assistido por IA para garantir:
- rastreabilidade de decisoes,
- reproducao de resultados,
- auditoria tecnica.

## Template recomendado
- Data/hora
- Objetivo do prompt
- Prompt enviado
- Arquivos afetados
- Resultado esperado
- Riscos identificados

## Registro atual (resumo)
- Criacao de scaffold da plataforma V4
- Geracao de ETL com modo sample/db
- Integracao de dados legados
- Implantacao de autenticacao e perfis
- Evolucao da pagina Admin para painel de gestao
- Evolucao de UX (tema escuro, dashboard, exportacao)

## Boas praticas
- Evitar prompts vagos para alteracoes criticas.
- Sempre registrar o objetivo funcional antes do prompt.
- Associar cada prompt a um checkpoint tecnico.
